Generated Examples

Here are some examples of performances generated by our model. These are selected randomly, as opposed to being cherry-picked. We also include several ground truth examples from the held-out test dataset.

All generations are unprompted, meaning the model only sees the associated Genre and Start tokens before generation begins. Examples are included both with and without the DPO-P loss.

All examples are rendered using the BBC Symphony Orchestra Piano library from Spitfire Audio.
